Explanation of CSS Selectors

Universal Selector (*) — Applies to all elements on the page.

Class Selector (.classname) — Used for multiple elements sharing the same style.

ID Selector (#idname) — Used for styling a single, unique element.

Descendant Selector (parent child) — Styles child elements within a specific parent.

Group Selector (a, b, c) — Groups multiple selectors to apply the same styling together.
